# Wave AJ — Bootstrap-Preparation-Checklist (BOO-104)

> 🌐 **Language:** English (this file) · [🇩🇪 Deutsch](wave-aj-bootstrap-prep.md)

**Problem before:** There was no customer-ready advance sheet mirroring the real `/bootstrap` interview questions. Appendix A covers only technical prerequisites; the questions lived only in `bootstrap/SKILL.md`; BOO-102 (integration-discovery) covers the deep integration part, not the interview.

**What is there now:** `docs/onboarding/bootstrap-prep.md` (+ `.en.md`) — mirrors 1:1 block A (10 questions), block B (6 + provider postflight), block C (doc architecture review) and block D (optional components) from `bootstrap/SKILL.md`. Per question: what is asked + who answers it on the customer side. Plus technical prerequisites (appendix A) and cross-reference to `integration-discovery.md` (BOO-102) + HANDBUCH §4.

**Effect:** The consultant can hand the sheet to the customer IT in advance („these questions are coming next week in the bootstrapping") — the session runs without retrieval loops. Pure documentation.
