## Stories
- **BOO-110** — dpo-Overview-Sketch.

## Aenderungen
- `dpo/overview.png` / `.en.png` (+ `.excalidraw`-Quellen).
- Zeigt: 3 Modi entlang des Lebenszyklus (ASSESS ← /ideation · REVIEW ← /implement · AUDIT katalog-getrieben) → Art. 5 DSGVO Grundprinzipien (7 Chips) → laenderspezifische Regelwerke (DSGVO/BDSG/nDSG) → Banner „AUDIT-Urteil: PASS · GAP · REVIEW-NEEDED — der Skill erfindet keine Rechtsberatung".

## Effekt
