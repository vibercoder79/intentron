# Wave AJ — Bootstrap-Vorbereitungs-Checkliste (BOO-104)

**Problem vorher:** Es gab keinen kundenfaehigen Vorab-Bogen, der die echten `/bootstrap`-Interview-Fragen spiegelt. Anhang A deckt nur technische Voraussetzungen; die Fragen lebten nur in `bootstrap/SKILL.md`; BOO-102 (integration-discovery) deckt den tiefen Integrations-Teil, nicht das Interview.

**Was jetzt da ist:** `docs/onboarding/bootstrap-prep.md` (+ `.en.md`) — spiegelt 1:1 Block A (10 Fragen), Block B (6 + Provider-Postflight), Block C (Doku-Architektur-Review) und Block D (Optional-Komponenten) aus `bootstrap/SKILL.md`. Pro Frage: was gefragt wird + wer es kundenseitig beantwortet. Plus technische Voraussetzungen (Anhang A) und Querverweis auf `integration-discovery.md` (BOO-102) + HANDBUCH §4.

**Effekt:** Der Berater kann den Bogen vorab an die Kunden-IT geben („diese Fragen kommen naechste Woche im Bootstrapping") — die Session laeuft ohne Rueckholschleifen. Reine Doku.

Spec: `specs/BOO-104.md`. Release: v0.7.1.
