Third onboarding document `docs/onboarding/artefakt-landkarte.md` (DE+EN): bridges the gap between the discovery questionnaires (`bootstrap-prep`, `integration-discovery`) and the phase artifacts. Master matrix across all 13 skills/phases with 7 rule sinks (`CLAUDE.md`, `CONVENTIONS.md`, `ARCHITECTURE_DESIGN.md §5`, `SECURITY.md`, `dpo/controls/`, `.claude/environment.json`, `DEVELOPER_ONBOARDING.md`) and a sign-off role per artifact. DESIGN.md pattern: frontend design is architecture (`§5` → `DESIGN.md`). Bootstrap step 4.3c seeds a filtered `solution-artefakte.md`; `migrate_boo_108()` for existing projects. HANDBUCH Appendix Z (DE+EN) + README onboarding section. Sketch `docs/assets/onboarding-flow.png`/`.en.png`, built in the render loop + visually checked.

Goal: an autonomous team of 3–4 people then develops compliantly on its own — contributing to the IT transformation.
